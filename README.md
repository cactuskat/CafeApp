# CafeApp

'A Brand Brew Day' is a simple two-screen cafe app. The home screen welcomes visitors with the name, an image of the location, and a short paragraph about the shop, plus a button that takes them to the menu. The menu screen lists the cafe's drinks with their prices in a clean, scrollable layout. An image of one of the featured drinks appears.

### Design Choices:
- Consistent color pallete of earthy tones: pleasing UI color pallete
- Kept the button location consistent across both screens: user has to think less
- Images are large and towards the center of the screen: directly grabbing focus
- Names are creative to be memoriable
- Menu is in ListView for clean code

### How to run it:
- Open 'cafeapp' folder on VS code
- Select 'Start Debugging'
- Select simulator of choice (Was tested with Google Chrome)
